================================================================================
                            QUICK START
================================================================================

1. Enable Mock Mode:
   export MOCK_MODE=true

2. Import Service Factory:
   const { getService } = require('./utils/serviceFactory');

3. Get Services:
   const anthropic = getService('anthropic');
   const gmail = getService('gmail');
   const drive = getService('drive');
   const upwork = getService('upwork');
   const calendly = getService('calendly');

4. Use Services:
   const response = await anthropic.createMessage({type: 'writer'});
   const doc = await drive.createDocument({name: 'Test', content: '...'});
   const jobs = await upwork.searchByNiche('technology', 5);

See MOCK_PROVIDERS_QUICKSTART.md for more examples.
